## Data

The dataset used to inform the inventory simulation in this project is publicly available and synthetic.

### Source

- **Kaggle Dataset (Pizza Sales):**  
  https://www.kaggle.com/datasets/shilongzhuang/pizza-sales

- **Original Challenge Context (Maven Analytics):**  
  https://mavenanalytics.io/blog/maven-pizza-challenge

This dataset was originally provided by Maven Analytics as part of an analytics and BI dashboard challenge and subsequently made available on Kaggle.

### Reproducing the Analysis

To run or reproduce this analysis, complete the following steps:

1. Visit the Kaggle link above and download the dataset (`pizza_sales.csv`).
2. Save the downloaded dataset file into this `data/` directory. (i.e., `data/pizza_sales.csv`)
3. Ensure your local environment satisfies the dependencies listed in `requirements.txt`.
4. Run the notebook in the `notebooks/` directory to execute the full analysis workflow and use the simulation logic implemented in `src/newsvendor_simulation.py`.
